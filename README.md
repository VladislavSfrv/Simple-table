# simple-table

Упрощённая версия проекта **smart-table** для первого знакомства с интерактивными таблицами.

## Запуск

```bash
npm install
npm run dev
```

## Чем проще, чем smart-table

| simple-table | smart-table |
|---|---|
| Один файл `main.js` | Много модулей в `components/` и `lib/` |
| Плоский массив данных | Связанные customers, sellers, purchase_records |
| Обычная HTML-таблица | Шаблоны `<template>` и `cloneTemplate` |
| Объект `state` | `FormData` и `processFormData` |
| Простые функции `filter` / `sort` | Компараторы `createComparison` и правила |
| Простая пагинация (prev/next) | Полная пагинация с кнопками страниц |

## Что делает приложение

1. **Отрисовка** — показывает строки из массива
2. **Поиск** — ищет по всем колонкам
3. **Сортировка** — клик по заголовку колонки
4. **Фильтры** — отдельные поля под каждой колонкой
5. **Пагинация** — разбиение на страницы

## Как перейти к smart-table

После того как разберётесь с этим проектом, в **smart-table** нужно будет:

1. Разбить логику на модули (`table.js`, `sorting.js`, `filtering.js`…)
2. Использовать HTML-шаблоны вместо `innerHTML`
3. Собирать состояние из формы через `FormData`
4. Нормализовать данные из нескольких таблиц (`initData`)
5. Вынести сравнение в `createComparison` с правилами

Каждая функция здесь (`applySearch`, `applySort`, `applyFilters`, `applyPagination`) — это прямой аналог соответствующего модуля в smart-table.

## Задания для практики

1. Добавьте колонку «ID» в таблицу
2. Сделайте сортировку по умолчанию по дате (новые сверху)
3. Добавьте подсчёт суммы на текущей странице
4. Сохраняйте состояние фильтров в `localStorage`
5. Перейдите к smart-table и найдите, где реализована каждая из этих возможностей
