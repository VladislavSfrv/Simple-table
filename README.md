# simple-table

## Запуск

```bash
npm install
npm run dev
```
## Что делает приложение

1. **Отрисовка** — показывает строки из массива
2. **Поиск** — ищет по всем колонкам
3. **Сортировка** — клик по заголовку колонки
4. **Фильтры** — отдельные поля под каждой колонкой
5. **Пагинация** — разбиение на страницы

## Задания для практики
1. Доделайте компоненты таблицы.
2. Добавьте колонку «ID» в таблицу
3. Сделайте сортировку по умолчанию по дате (новые сверху)
4. Добавьте подсчёт суммы на текущей странице
5. Сохраняйте состояние фильтров в `localStorage`
